# SMPC

A curated research and learning repository focused on **Secure Multiparty Computation (MPC)**.

## Purpose

This repository exists to document and organize practical and theoretical knowledge about MPC, including:

- Core definitions and threat models
- Security notions (semi-honest, malicious, covert, abort/fairness/composability)
- Major protocol families (Yao, GMW, BGW/CCD, SPDZ, HE/FHE hybrids)
- Performance trade-offs (communication, computation, rounds, offline/online split)
- Real-world frameworks and deployments
- Recent advances, risks, and implementation considerations

## Main Content

- `SMPC.md`: a long-form, structured technical brief on MPC.

## Who This Is For

- Engineers evaluating privacy-preserving computation systems
- Students and researchers studying modern MPC
- Practitioners comparing protocol families and tooling choices

## Repository Goal

Provide a single, readable reference that bridges cryptographic foundations and deployment-oriented engineering decisions for secure computation.
